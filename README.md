# 一个基于51串口收发小实验装置

1. **设备**：STC8G1K08A
2. **功能**：
- 定时发送指定空闲包
- 根据拨码开关状态，发送不同的功能包
3. **测试平台**：*特定*上位机

## 项目结构
code中存放的是项目软件，可用stc-isp烧录 \
hardware中存放的是电路部分，用嘉立创EDA打开。 

## 物料清单
0603 10kΩ 电阻 3个 \
0603 led 1个 \
STC8G1K08A  1个 \
卧插拨码开关 1个 \
弯针排针（母） 1个 \
去嘉立创打块板子 1个 \
